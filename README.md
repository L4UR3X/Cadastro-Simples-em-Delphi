# Cadastro Simples em Delphi (VCL)

Projeto simples desenvolvido em **Delphi 13 (Athens)** com o objetivo de praticar a **sintaxe do Object Pascal**, o **modelo orientado a eventos** e o uso de **formulários VCL**.

---

## Funcionalidades

- Formulário com campos de **Nome** e **Email**
- Validação de campos obrigatórios
- Botão **Salvar** para exibir os dados em um `TMemo`
- Botão **Limpar** para resetar o formulário
- Inicialização do formulário com o memo vazio

---

## Tecnologias utilizadas

- **Delphi 13 (Athens)**
- **Object Pascal**
- **VCL (Visual Component Library)**

---

## Conceitos aplicados

- Estrutura de um projeto Delphi
- Programação orientada a eventos
- Manipulação de componentes visuais (`TEdit`, `TButton`, `TMemo`)
- Validação de dados em memória
- Uso de métodos e procedures
