# Cadastro Simples em Delphi

Projeto de estudo desenvolvido em **Delphi 13 (Athens)** para praticar **Object Pascal**, uso de **formulários VCL** e **eventos**.

---

## Funcionalidades
- Formulário com campos Nome e Email
- Validação de campos obrigatórios
- Exibição dos dados em um `TMemo`
- Botão para limpar o formulário

---

## Tecnologias
- Delphi 13 (Athens)
- Object Pascal
- VCL
